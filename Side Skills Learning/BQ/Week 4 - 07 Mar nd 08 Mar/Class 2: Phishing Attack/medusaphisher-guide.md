# Medusa Phisher Guide

1. Open terminal in linux (Kali)
> github clone https://github.com/Adrilaw/MedusaPhisher.git
2. After successfully cloning
> cd MedusaPhisher (use ls -la to check the correct and full name of file)
3. Run
> chmod +x setup.sh medusa_phisher.sh
<img width="823" height="323" alt="image" src="https://github.com/user-attachments/assets/c515b37e-378d-4c47-81e4-f3138970b1d1" />

4. Then 
> ./setup.sh (to install all required dependencies)
5. The run
> ./medusa_phisher.sh

And this time it should generate the link.
