# Question 1:
Write down the function (codes) of all insertion and deletion scenarios of singly link list. 
Also perform dry run of line by line execution of code.
⦁Insertion Scenarios
⦁	Insertion at the beginning (Head insertion)
⦁	    Add a new node before the current head.
⦁	    The new node becomes the new head.
⦁	Insertion at the end (Tail insertion)
⦁	    Add a new node after the last node.
⦁	    The new node becomes the new tail.
⦁	Insertion at a specific position (Middle insertion)
⦁	    Add a new node after a given node or at a specific index in the list.
